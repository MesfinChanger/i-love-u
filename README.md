## 🚀 One-Click Global Publishing

I Love U is optimized for high-performance hosting on Firebase and mobile support via Capacitor.

### Steps to Publish:

1.  **Production Credentials**: 
    - Verify Firebase configuration in `src/firebase/config.ts`.
    - Provide a valid `NEXT_PUBLIC_FIREBASE_API_KEY` in your environment (see `.env`).
    - Set `NEXT_PUBLIC_BASE_URL` to your production domain: `https://studio-9260674464-8df20.web.app`.

2.  **Deploy to Web**: 
    ```bash
    npm run build
    npm run deploy
    ```
    Your app will be live at: `https://studio-9260674464-8df20.web.app`

3.  **Mobile Support (Android/iOS)**:
    Build your native binaries by running:
    ```bash
    npm run build
    npx cap sync
    ```
